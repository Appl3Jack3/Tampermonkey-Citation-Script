# Tampermonkey-Citation-Script
A customer citation generator designed for fast accurate webpage citations directly inside the broswer. This script adds a floating APA button, an Alt + Q hotkey, auto-detects webpage details, supports multiple authors, and stores a citation history sidebar for quick reuse. Built for students, researchers, and anyone who cites online sources.
