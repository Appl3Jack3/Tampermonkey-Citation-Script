# Code of Conduct

All users and contributors must interact respectfully and professionally. Harassment, abuse, discrimination, or hostile behavior will not be tolerated.

The project owner reserves the right to block or remove individuals who violate this policy.

This tool does not generate academic content and does not assist with quizzes, exams, or assignments. It only formats citations provided by the user and is intended to support proper academic referencing in accordance with SNHU academic integrity guidelines.
